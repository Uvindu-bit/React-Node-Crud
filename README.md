# React-Node-Crud

## Installing NPM

######  Installing NPM in client side 
> Go to the client folder

```
npm install
```

###### Installing NPM in server side
> Go to the server folder

```
npm install
```

> And the installation is done

## Starting the application

###### To start client side

```
npm start
```

###### To start server side

```
node index.js
```
